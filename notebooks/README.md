# Notebooks
This directory is used to store all notebooks. Notebooks should be sorted into
further directories based on what the notebook is testing.
(i.e. a group of notebooks testing specific API functionality should be grouped into a
subfolder)
```
.
├── notebooks/             # Jupyter Notebooks
│   ├── open-cv/           # Notebooks testing OpenCV
│   ├── video-stream/      # Notebooks testing live video streaming
│   └── ...                # etc.
```
